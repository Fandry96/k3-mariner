## 2026-03-25 - Contextual Empty States for Critical Inputs
**Learning:** Users often abandon setups when faced with an empty, required API key field, even if a tooltip is present. Combining a standard markdown help tooltip with a conditionally displayed, directly clickable `st.caption` link when the input is empty significantly reduces abandonment. Additionally, using `format_func` in `st.selectbox` allows presenting user-friendly model names without altering underlying technical IDs.
**Action:** Always conditionally surface direct, clickable resolution links for empty critical setup inputs, and use `format_func` for technical identifiers in selectboxes.
